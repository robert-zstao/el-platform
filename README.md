# em-platform
This is a multi-function management platform based on SpringBoot
## 基本介绍
#### 这是一个基于SpringBoot开发的多功能企业综合管理平台，主要具有以下功能：
* 基本的日常管理功能，包括部门，人员，组织架构，打卡记录等
* 开发管理平台，包括数据库管理，前后端代码生成。
* 平台监控
* 服务器管理
* 日志操作记录
* 办公任务推送、提交
